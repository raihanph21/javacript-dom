Document Object Model (DOM)

DOM : representasi elemen HTML pada dokumen menjadi sebuah objek. representasi/permodelan/memodelkan elemen html menjadi sebuah objek.

  • Antarmuka Pemrograman berbasis objek yang merepresentasikan dokumen web
  • DOM membuat seluruh komponen dari halaman web dapat diakses dan dimanipulasi
  • komponen:
    - Elemen HTML
    - Atribut
    - Text
    - dll
  • DOM dapat dimanipulasi (dibuat, diubah, dihapus) menggunakan javascript

Struktur Hierarki DOM

  • Root Node
    - Node yg menjadi sumber dari semua node lain di dalam DOM
    - document
  • Parent Node
    - node yg berada 1 tingkat diatas node lain
    - body adalah parent dati h1,p,p,div, dan a
  • Child Node
    - node yg berada 1 tingkat dibawah node lain
    - h1 dari adalah child dari body
  • Children
    - child node yg bertipe elemen html
  • Siblings
    - child node yg memiliki parent yg sama
